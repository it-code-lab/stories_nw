Suno.com song + Video Clip to video generator

  1. Place song audio.wav inside edit_vid_audio.
  2. Place video clip (5/8 secs) at edit_vid_input. (Change slow down factor below if needed)
  3. Click on below button
  4. This would make longer clip by slowing speed and then create video by repeating to audio length It will then place the video as composed_video.mp4 inside the root folder. And will place the input audio file as audio.wav inside the root folder. Then it will call script to make WordTimeStamps file.
  5. After processing is done, review the captions at Open Caption Editor (Opens in new tab) and make necessary change.
  6. Then Run OBS Recorder to create final video for upload. Parameters: Caption-style1, words per caption-10,outputfile - edit_vid_output/output.mp4

Suno.com song + Images to video generator

  1. Place song audio.wav inside edit_vid_audio.
  2. Place images in edit_vid_input. (If needed, change input image size and clip duration for each image )
  3. Click on below button
  4. This would make video clips and then create video by repeating to audio length It will then place the video as composed_video.mp4 inside the root folder. And will place the input audio file as audio.wav inside the root folder. Then it will call script to make WordTimeStamps file.
  5. After processing is done, review the captions at Open Caption Editor (Opens in new tab) and make necessary change.
  6. Then Run OBS Recorder to create final video for upload. Parameters: Caption-style1, words per caption-10,outputfile - edit_vid_output/output.mp4
